Documentation

Regim documentation help provides a series of indications and specifications in order to help our users understanding all the software features.

Displacement registration method

From the image registration methods provided by SimpleITK, Regim was responsible for adjusting them in the most optimal way to obtain the best results.
A registration method use several components which together defines a registration instance. Displacement registration method uses the following components:

  • Transformation
    • Affine Transform
  • Similarity Metric
    • Mutual Information
  • Optimizer
    • Gradient Descent
  • Interpolator
    • Linear

Resctrictive registration method

From the image registration methods provided by SimpleITK, Regim was responsible for adjusting them in the most optimal way to obtain the best results.
A registration method use several components which together defines a registration instance. Restrictive registration method uses the following components:

  • Transformation
    • Similarity 2D Transform
  • Similarity Metric
    • Mutual Information
  • Optimizer
    • Regular Step Gradient Descent
  • Interpolator
    • Linear

Iterations

The iterations data field indicates the number of iterations that the algorithm took to achieve a certain similarity value, hence, the more differences present the images, the more iterations the algorithm will need to perform, therefore, it will require more processing time.

Metric value

Metric value data field is very important when determining the degree of success that the process had.
The measure used gives values, approximately, between 0 and -2, where -2 is the best possible result. The bar shown in the software represents the percentage of success.
Below are ranges that explain the perfomance according to the metric (t) :

  • Low:   t   <   40%
  • Medium:   40%   <   t   <   70%
  • High:   t   >   70%

Maximum iterations

The maximum iteration input specifies the maximum number of iterations taken by the algorithm. This can limit the system's operation if the specified value is too small.

Recommended value   > =   100
